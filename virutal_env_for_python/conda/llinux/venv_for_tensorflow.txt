# To install/setup tensorflow in conda virtualenv in ubuntu:

    1. At first u should have already setup miniconda/anaconda in you pc.
        (conda package manager should already be setup in your laptop)

    
    2. Open a terminal and create a new conda virtual environment.
        SYNTAX: ==> conda create -n venv python=version
        eg: ==> conda create -n tfenv python=3.10


    3. Activate the above created conda virtual environment.
        ==> conda activate tfenv


    4. Install 'Jupyter Notebook/JupyterLab' and 'ipykernel'

        ==> conda install jupyter ipykernel
        OR
        ==> conda install jupyterlab ipykernel


    5. Install tensorflow framework (CPU Version)


        ==> conda install tensorflow
        OR
        ==> conda install tensorflow=2.12
        OR
        ==> conda install -c conda-forge tensorflow=2.5
    

    6. Create a virtual environment's python kernel to use for Juputer.
        SYNTAX: ==> python -m ipykernel install --user --name=myenv --display-name "venv"
        eg: ==> python -m ipykernel install --user --name=myenv --display-name "tfenv"

    

    7. Launch/Run Jupyter server in 8888 port

        ==> jupyter notebook
        OR
        ==> jupyter lab
    


    8. To verify successful tensorflow installation:

        write following code in notebook cell and execute cell:

            import tensorflow as tf
            print("Tensorflow version: ", tf.__version__)

        
        --> Hopefully, u should see latest tensorflow version installed.
